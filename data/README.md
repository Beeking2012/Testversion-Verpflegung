# Cloud App - Datenmanagement

Eine Single-Page-Application zur Verwaltung von Cloud-Daten mit vier Hauptfunktionen:

## Seitenübersicht

1. **Auswahl** - Datenauswahl basierend auf Cloud-Daten
2. **Cloud-Abgleich** - Vergleich der Auswahl mit Cloud-Daten
3. **Berechnung** - Berechnungen und eigene Items
4. **Verwaltung** - Cloud-Elemente Verwaltung

## Technologie

- Reines HTML, CSS, JavaScript
- Local Storage für Offline-Funktionalität
- Cloud-API Integration
- Single-Page Application (SPA)

## Entwicklung

```bash
# Repository klonen
git clone https://github.com/your-username/cloud-app.git

# In Verzeichnis wechseln
cd cloud-app

# Lokalen Server starten (z.B. mit Python)
python -m http.server 8000

# Oder mit Node.js
npx serve .
